    Just Another Fine Pipeline Language

    This is a pipeline framework. It’s built on top of Akka actors to support arbitrary data flow pipelines. It is wholly agnostic to the kinds of data that flow through the pipelines and the implementations of the steps that process them.

    See http://jafpl.com/ for more general information.

    Create a new instance of Jafpl.

    Create a new instance of Jafpl.

    This method attempts to instantiate a class to configure Jafpl using the system property name com.jafpl.config.JafplConfigurer. If no such property exists, the default configurer is instantiated.

    A new Jafpl instance is the instantiated and passed to the configurer. After configuration, the instance is closed; attempting to update the properties will raise an exception.
